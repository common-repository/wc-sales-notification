== Description ==
WC Sales Notification is a plugin for Showing your orders or fake notification on your website. You can show the notification all pages in your website. It gives you awesome features, is to generate for fake notifications. The plugin Has Wonderful customization option and incoming and outgoing animation options.

You can see the plugin demo here : [Live Demo](https://woolentor.com/demo/sales-notification/)

== Features: ==
* Real sales notification
* Notification limit
* Single Blog Page Builder
* Notification loading time
* Real sales notification
* Notification limit
* Notification loading time
* Manual notification
* Notification showing position
* Notification image position
* Time interval each notification
* Sales upto date option
* Incoming animation option
* Outgoing animation option
* Background color option
* Heading color option
* Content color option
* Cross icon color option

== Installation ==
This section describes how to install the WC Sales Notification Plugins for WordPress get it working.

= 1) Install =

= Install: =
1. Go to the WordPress Dashboard "Add New Plugin" section.
2. Search For "WC Sales Notification".
3. Install, then Activate it.

= OR: =
1. Unzip (if it is zipped) and Upload `wc-sales-notification` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
